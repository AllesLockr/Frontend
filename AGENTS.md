# AGENTS.md ‚Äì AllesLocker Frontend

## Commands

| Command | What it does |
|---|---|
| `pnpm dev` | Start Vite dev server (proxies `/api` ‚Üí `http://localhost:8080`) |
| `pnpm build` | `tsc -b && vite build` (typecheck both tsconfig project refs, then build) |
| `pnpm lint` | ESLint on `.` |
| `pnpm format` | Prettier --write on `**/*.{ts,tsx}` |
| `pnpm typecheck` | `tsc --noEmit` standalone |
| `pnpm openapi-ts` | Re-generate API client from OpenAPI spec at `http://localhost:8080/v3/api-docs` |

## Architecture

- **Entry**: `src/main.tsx` ‚Üí `<BrowserRouter>` > `<QueryClientProvider>` > `<ThemeProvider>` > `<App>`
- **Routing**: `src/App.tsx` defines routes; every page except `/login` is wrapped in `<ProtectedRoute>` which redirects unauth users
- **Auth**: `src/context/AuthContext.tsx` ‚Äî stores JWT in `localStorage`, sets Bearer header on the generated fetch client (`client.gen.ts`)
- **API client**: `src/client/` is **auto-generated by `@hey-api/openapi-ts`** and blocked by `.gitignore` ‚Äî never commit it. Always re-generate after backend changes
- **Proxy**: Vite dev server proxies `/api` ‚Üí `http://localhost:8080` (see `vite.config.ts`)
- **Theme**: Custom `ThemeProvider` (not next-themes) in `src/components/theme-provider.tsx`; press `d` to toggle dark/light. Persists to `localStorage`
- **UI components**: shadcn/ui in `src/components/ui/` (Radix Vega style); dialogs in `src/dialog/`; pages in `src/page/`
- **CSS**: Tailwind CSS v4 via `@tailwindcss/vite` plugin; `src/index.css` imports tailwind, tw-animate-css, shadcn tailwind, and Inter font

## Conventions & gotchas

- **Code style**: no semicolons, double quotes, 4-space indent (enforced by Prettier)
- **Path alias**: `@/` ‚Üí `src/` (Vite resolve + tsconfig `paths`)
- **Imports**: `verbatimModuleSyntax: true` ‚Äî must use `import type` for type-only imports. Always include `.tsx`/`.ts` extensions in relative imports
- **Class merging**: use `cn()` from `@/lib/utils` (clsx + tailwind-merge)
- **TypeScript strict**: `noUnusedLocals`, `noUnusedParameters`, `strict: true` turned on
- **Prettier**: v3 with `prettier-plugin-tailwindcss` auto-sorts classes; `tailwindFunctions: ["cn", "cva"]`
- **pnpm workspace**: `esbuild` and `msw` builds explicitly disabled in `pnpm-workspace.yaml`
